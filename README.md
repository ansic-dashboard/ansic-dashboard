# 안식(ANSIC) 매출 대시보드 v3.1

한옥 파인다이닝 안식의 매출·운영·마케팅 통합 분석 대시보드.

## 5개 탭 구성
1. **통합 요약** — 월 목표(5천만) 진행, 핵심 KPI(매출·고객·팀수·객단가·회전율) 전월/작년 대비, 월별 매출 막대그래프(2024·2025·2026 나란히), 30일 추이 꺾은선, 마케팅 비용·활동 요약, 인플루언서 상태 자동 집계, 자동 인사이트
2. **매출 분석** — 연/월 선택(2024~), 10일/20일/한달/현재 기간 탭, 현재·전월·작년 비교, 운영지표(인원·팀수·객단가·회전율), 예약 유입경로(전화·캐치·네이버), 요일별 평균, 일별 매출 일람(점심·저녁·예약·워크인·배달·회전율), 원인분석형 인사이트
3. **인플루언서 성과** — 통합 요약(비용·인원·상태 자동집계·ER 비교) + 월별 조회(엑셀형 표, 행마다 직접 수정, 상태 드롭다운→통합요약 자동 반영)
4. **마케팅 효과** — 핵심 요약(편집), 월별 총매출 비교(2026 실시간 재계산·2025 케이터링 보정), 케이터링 조정 내역, 전월 대비 증감폭, 비용 내역, 참고·주의할 점(편집)
5. **시간대별 매출 관리** — OKPOS 엑셀(연간매출표 .xlsx / P&L .xlsx / 시간대별 점심·저녁 .xls)을 드래그&드롭 업로드 → 자동 파싱·분석. 점심/저녁 비중, 시간대별(11~21시) 매출·피크, 요일×시간대 히트맵, 요일별 객단가. 핵심 결과는 "매출 분석" 탭에도 노출. 업로드 시 monthly_sales·hourly_sales 전체 교체(delete-then-insert). 객단가 = POS매출 ÷ 총방문인원

## 데이터 출처
- **매출**: 2024~2026.03 = `public/data/historical.json` / 2026.04~ = 안식 P&L 시트 실시간 fetch (110행 일계매출)
- **운영지표**: 2024~2026.03 = 매출표 엑셀(연도별 행 구조 다름) / 2026.04~ = P&L 현황 시트 → `public/data/operational.json` (863일)
- **마케팅·인플루언서**: Supabase (사이트에서 직접 수정 가능)
- **시간대별 매출**: 직원이 OKPOS 엑셀을 업로드 → SheetJS 서버 파싱 → Supabase(monthly_sales/hourly_sales). xls(구버전)·xlsx 모두 지원

## 환경변수 (Netlify)
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `OPENWEATHER_API_KEY` (선택 — 없으면 날씨 기능만 비활성, 나머지 정상 작동)

## 편집 정책
- **매출 숫자만 잠금** (자동 데이터)
- 인플루언서 정보·상태·인사이트, 마케팅 요약·참고·주의 메모는 모두 사이트에서 수정 → Supabase 저장
