# 안식 매출 대시보드 v2.1

ANSIC Restaurant Sales & Marketing Analytics Dashboard

## 🆕 v2.1 변경 사항

### 통합 요약 탭 — 완전 재구성
- **이번 달 목표 진행 카드** (5천만원 기준, 진행률 바, 남은 매출/일수/하루 필요액/월말 예상)
- 이번 주 / 작년 동기간 비교 (자연어로 풀어쓰기)
- 30일 매출 추이 (2024/2025/2026 비교, 선 굵게)
- **마케팅 캠페인 타임라인** (가로 형태, 완료/진행중/예정 색상 구분)
- **행동 제안형 인사이트** (단순 수치 → "왜" + "그래서 뭘 할지")
- "이번 달 마케팅 효과 328만원" 카드 제거 (오해 위험)

### 매출 분석 탭 — 월 선택 + 구간 탭
- **연/월 드롭다운 선택** (모든 데이터가 있는 월 선택 가능)
- **10일 / 20일 / 말일 / 현재** 탭 (참고 페이지 구조 차용)
- 전월 동기간 / 작년 동기간 비교 카드 + 자연어 설명
- 요일별 평균 매출 차트 (일요일 빨강, 토요일 파랑 색상 구분)
- 일별 매출 일람표 (점심/저녁 분리)

### 마케팅 효과 탭 — 엑셀 그대로
- 월별 매출 + 전월 대비 증감폭 (2024/2025/2026 3개년)
- **케이터링 조정 후 참고 비교** 부활
- 마케팅 비용 타임라인 (2/27 메타1차 → 4/14 메타2차 → 4/16 인플4월 → 5/23 인플5월)
- 비용 내역 상세 (촬영비/광고비/관리비 분리)
- 핵심 요약은 클릭해서 직접 수정 가능
- "자연 평균" → "직전 2년 평균 변화율" 명확한 표현

### 인플루언서 탭 — 라벨 명확, 영상 링크 부활
- **상태 드롭다운**: 촬영 전 / 촬영 완료 / 편집 중 / 업로드 완료
- **촬영일 필드** 추가
- **영상 보기 하이퍼링크** 부활 (▶ 영상 보기 → 새 탭)
- 강점 / 주의점 / 재계약 의견 명확한 색상 구분
- 인사이트 입력 시 각 필드 옆에 친절한 설명 ("도달 (영상 본 사람 수)")

### 전반적 개선
- 글자 색 더 진하게 (#0F172A → #000)
- 핑크색 완전 제거 (BI 보존하되 분석 도구는 가독성 우선)
- 모든 폰트 weight bold/extrabold 강화
- 한국어 날짜 형식 "5월 21일 (목)" 표시

## 🚀 배포 방법

### 1. Supabase 마이그레이션 (v2.0 → v2.1)

새 필드 추가 (`shoot_date`, `status`)가 필요합니다.

기존 데이터 유지하면서 업그레이드:
1. Supabase → SQL Editor
2. `supabase/schema.sql` 내용 전체 실행
3. 이미 있는 데이터는 무시되고 새 컬럼만 추가됨

### 2. GitHub에 푸시

전체 파일 통째로 덮어쓰기 권장.

### 3. Netlify 자동 빌드

환경변수는 v2.0 그대로 사용:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
